# Customer Behavior Data Analysis

This project explores customer demographic and spending behavior using exploratory data analysis techniques.

## Objective

Understand how demographic variables such as age, income, and work experience influence customer spending behavior.

## Dataset

The dataset includes customer demographic information and spending patterns.

Variables include:

- Age
- Income
- Work Experience
- Spending Score

## Analysis Performed

Exploratory Data Analysis (EDA):

- Summary statistics
- Distribution analysis
- Boxplots
- Scatterplots
- Relationship analysis between income, age, and experience

## Key Insights

- Income is strongly associated with spending behavior
- Age and experience show moderate relationships with income
- Spending patterns vary significantly across demographic groups

## Tools Used

Python  
Pandas  
Seaborn  
Matplotlib
